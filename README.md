# PodoArta
Пет-проект "сайт подологического центра"

### Функционал
- Просмотр информации об услугах постранично
- Сортировка услуг по категориям
- Отправка писем с вопросами/отзывами на почту
- Прямая связть через мессенджеры(whatsapp, telegram ,vk)

## Deploy
<a href="https://podo-arta.vercel.app/" target="_blank">podoarta.com</a>

## Технологии
- TS
- React
- JS
- CSS

## Разработка
Для установки и запуска проекта, необходим NodeJS v8+.

### Настройка проекта
```sh
$ npm i
$ npm run dev
```
